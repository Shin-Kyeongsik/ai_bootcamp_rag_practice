# 일차별 실습 안내

이 문서는 각 일차별 실습 주제와 목표, 결과물을 안내합니다. 매일 실습을 통해 하나의 결과물 파일을 완성하며, 5일차에는 팀별 최종 데모 발표로 마무리합니다.

## 일차별 실습 내용

| 일차  | 주제                        | 목표                                                                 | 결과물                             |
|-------|-----------------------------|----------------------------------------------------------------------|-----------------------------------|
| 1일차 | Agent와 일반 LLM의 차이 이해 | 사용자의 요청을 단계별 작업으로 분해하는 Agent 흐름 이해             | `agent_workflow_notes.md`         |
| 2일차 | RAG 기본 실습               | 부트캠프 문서를 검색해서 질문에 답하는 구조 만들기                   | `rag_answer_examples.md`          |
| 3일차 | RAG 답변 품질 개선          | 검색된 문서 조각을 바탕으로 더 정확한 답변 만들기                    | `rag_quality_improvement_notes.md`|
| 4일차 | 안내 AI 응답 흐름 설계      | 참가자 질문 유형에 따라 적절한 문서를 검색하고 답변하는 흐름 만들기 | `guide_ai_response_flow.md`       |
| 5일차 | 최종 데모와 발표            | 팀별 안내 AI 사용 시나리오 발표                                      | 5분 데모 발표 자료                |

## 팀 프로젝트 조건

- 최종 프로젝트는 **3~4인 1팀**으로 진행합니다.
- 각 팀은 **최소 5개의 참가자 질문**에 대해 **원본 문서 근거를 포함한 답변**을 시연해야 합니다.
- 최종 발표 시간은 **팀당 5분**이며, 질의응답은 **2분**입니다.
