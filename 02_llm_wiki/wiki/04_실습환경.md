# 실습 환경 설치 가이드

이 문서는 부트캠프 참가자가 실습 전에 준비해야 할 개발 환경을 안내합니다. 원활한 실습을 위해 부트캠프 시작 전에 아래 준비물을 미리 설치해 두는 것을 권장합니다.

## 필수 준비물

- 개인 노트북
- 전원 어댑터
- Chrome 브라우저
- GitHub 계정
- ChatGPT 또는 Claude 계정
- VS Code
- Python 3.11
- Git
- Docker Desktop

## 권장 사양

- **운영체제**: macOS 13 이상, Windows 11 이상, Ubuntu 22.04 이상
- **메모리**: 최소 8GB, 권장 16GB 이상
- **저장공간**: 최소 여유 공간 10GB

## 설치 지원 정보

- **Docker 설치 지원**: 2026년 8월 4일 화요일 10:30 ~ 11:00, A동 3층 Lab 301 앞 멘토 데스크
- **Python 버전**: 3.11 권장 (3.12 사용자는 호환성 문제 확인 필요)
- **실습 저장소 배포**: 1일차 오후, GitHub Classroom 링크로 제공

## 문제 해결 안내

설치가 어려운 경우 다음 순서로 도움을 요청하세요:

1. 실습 환경 설치 가이드의 체크리스트 확인
2. 에러 메시지 캡처
3. 멘토링 시간(17:00 ~ 18:00)에 멘토에게 질문
4. Docker 설치 문제는 2일차 10:30 ~ 11:00 공식 지원 시간 활용
