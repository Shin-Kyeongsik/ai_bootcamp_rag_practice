## 2. 질문 방법

멘토에게 질문하기 전에 아래 순서를 먼저 진행하면 더 빠르고 정확하게 도움을 받을 수 있습니다.

1. 먼저 실습 문서의 **FAQ를 확인**합니다.
2. 같은 팀원과 **5분 이상 함께 문제를 확인**합니다.
3. **에러 메시지와 시도한 해결 방법을 정리**합니다.
4. **멘토 데스크에서 질문**합니다.

## 3. 질문 채널 (Slack)

온라인 질문은 부트캠프 전용 Slack 워크스페이스를 사용합니다.

| 채널명 | 용도 |
| --- | --- |
| `#bootcamp-qa` | 일반 질문 |
| `#bootcamp-notice` | 운영진 공지 |
| `#assignment-help` | 과제 관련 질문 |

- 일반 질문은 **`#bootcamp-qa`** 채널을 이용합니다.
- 운영진 공지는 **`#bootcamp-notice`** 채널에 올라옵니다.
- 과제 관련 질문은 **`#assignment-help`** 채널을 이용합니다.

## 4. 답변 정책

- 멘토는 **정답 코드를 그대로 제공하지 않습니다.**
- 대신 **에러 원인, 확인 순서, 참고 문서 위치**를 안내합니다.
- **과제 제출 마감 1시간 전(매일 22:59 이후)에는 긴급 질문만 우선 답변**합니다.
